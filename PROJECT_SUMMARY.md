# StaticCMS - Project Summary

StaticCMS is a lightweight, powerful Content Management System that combines a **FastAPI** backend with a **React** (Vite) admin dashboard to generate fully responsive, high-performance static websites.

## üöÄ Key Features

### 1. Block-Based Editor
- Create pages using modular blocks: **Hero sections**, **Text content**, and **Image Sliders**.
- Support for multiple sliders on the same page with unique initialization to prevent JavaScript conflicts.

### 2. Advanced Navigation (Header)
- **Dynamic Logo**: Support for custom image uploads or text-based branding.
- **Nested Menus**: Recursive sub-menu management (dropdowns) with pure CSS/Glassmorphism.
- **Configurable CTA**: A customizable "Call to Action" button in the header (manageable text, link, and color).
- **Glassmorphism Design**: Premium "Liquid Glass" header with background blur and translucency.
- **Responsive Navigation**: Mobile-first hamburger menu for smaller screens.

### 3. Static Site Generation (SSG)
- Instant generation of HTML/CSS/JS files from the database.
- Optimized for performance and SEO with metadata management.
- Integration of **Lucide Icons** and **Swiper.js** for modern UI/UX.

### 4. Media & API
- Integrated image upload system.
- Standardized API on port `8000` with correctly configured CORS for local development.

- **Backend**: Python, FastAPI, SQLAlchemy (SQLite), Jinja2 (Templating).
- **Frontend Admin**: React, Tailwind CSS, Axios, Lucide-React.
- **Generated Site**: Vanilla JS, Tailwind CSS, Swiper.js, Lucide Icons.

- `admin/`: React source code for the dashboard.
- `backend/`: FastAPI application, models, and schemas.
- `backend/templates/`: Jinja2 templates for site generation (`base.html`, `blocks/*.html`).
- `output/`: The generated static website files.
- `cms.db`: SQLite database storing pages, blocks, and menus.
